# Scoutnet-stuff

Här finns diverse skript, filer, och annat som relaterar till Scoutnet och kan
användas av användare och utvecklare i scoutkårer och arbetsgrupper inom
Scouterna. Ändringsförslag mottages tacksamt via Github Issues eller som pull
requests. Övriga frågor hanteras enklast via scoutnet@scouterna.se.
